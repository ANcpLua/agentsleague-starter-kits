# Agents League Starter Kits

Submissions for the **Microsoft Agents League Hackathon** — 3 competition tracks built with .NET 10.

## Tracks

| Track | Project | Description |
|-------|---------|-------------|
| **Track 1 — Creative Apps** | [Code Intelligence MCP Server](./track-1-creative-apps/) | 6 AI-powered tools for GitHub Copilot via MCP |
| **Track 2 — Reasoning Agents** | [CertPrepAgents](./track-2-reasoning-agents/) | 5-agent certification coaching pipeline |
| **Track 3 — Enterprise Agents** | [Enterprise Support Agent](./track-3-enterprise-agents/) | IT helpdesk for M365 Copilot Chat with connected agents |
